# Radar

https://github.com/a7medbehiry/Radar/assets/90977906/202bb6aa-dc47-4a88-b057-2acc5fd83ccd

A simple Radar build using esp8266, Ultrasonic Sensor, Servo Motor, BreadBoard, and leds.

# Components

	1 - BreadBoard
	2 - ESP8266
	3 - Ultrasonic Sensor
	4 - Servo Motor
	5 - Red Led
	6 - Green Led
